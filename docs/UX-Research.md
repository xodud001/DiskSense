# UX Research: Disk Scanner Apps

## 참고한 앱

| 앱 | 특징 | 장점 | 단점 |
|---|---|---|---|
| **DaisyDisk** | 라디얼 선버스트 | 한눈에 비율 파악, 드래그 앤 드롭 삭제, 프리뷰 | 파일 "왜 큰지" 설명 없음. 중첩 깊이 보기 불편. |
| **CleanMyMac X** | 카테고리 자동 정리 | 원클릭, 카드형 카테고리, Smart Scan | 블랙박스 (뭘 지우는지 불투명), 공격적 마케팅, 구독 피로 |
| **GrandPerspective** | 트리맵 | 정보 밀도 최고, 색상으로 타입 구분 | 못생김, 컨텍스트(이유)가 아예 없음 |
| **OmniDiskSweeper** | 컬럼 뷰 (Finder 스타일) | macOS 네이티브 익숙함, 폴더 드릴다운 빠름 | 아이콘/시각화 없음, 건조함 |
| **Disk Inventory X** | 트리맵 + 폴더 트리 | 두 관점 동시 제공 | 2005년 UI, 방치됨 |
| **CleanShot/Dropzone 류** | 간결한 메뉴바 | 접근성 좋음, 방해 없음 | 깊은 분석 불가 |

## 공통 장점 (채택할 것)

1. **라디얼/트리맵으로 한눈에 비율** — DaisyDisk / GrandPerspective
2. **카테고리 카드** (앱/개발/캐시 등 타입별 색) — CleanMyMac
3. **실시간 스트리밍 스캔 결과** — DaisyDisk (이미 우리 앱에 있음)
4. **컬럼 드릴다운** (폴더 → 하위) — OmniDiskSweeper
5. **메뉴바 미니 게이지** — CleanMyMac

## 공통 단점 (피할 것)

1. ❌ **"왜 지워도 되는지" 설명 없음** → AI 설명으로 차별화 (우리 핵심)
2. ❌ **블랙박스 원클릭 삭제** → 항상 승인 기반
3. ❌ **광고성 알림/배지** → 알림은 임계치 돌파 시에만
4. ❌ **폴더 사이즈 실시간 갱신이 무거워서 렉** → actor + 0.25s 스냅샷 (이미 적용)
5. ❌ **"정리 점수" 게이미피케이션 허세** → 숫자는 용량(GB)과 절약 가능량만
6. ❌ **유료 기능 T&C 팝업** → 없음

## DiskSense 고유 방향

각 앱의 강점을 조합하되, **"AI가 컨텍스트를 설명"** 을 메인 축으로 세운다.

### 핵심 원칙
- **정보 > 장식**: 모든 카드에 "왜 중요한가" 한 줄 설명이 있어야 함
- **투명성**: 지울 때 전체 경로 + 마지막 수정일 + 복구 가능성을 항상 보여줌
- **점진적 공개**: 게이지 → 카테고리 → Top N → 드릴다운, 스크롤 깊어질수록 상세
- **밀도와 공백의 균형**: CleanMyMac처럼 카드 펑펑 쓰지 않고, Finder처럼 빽빽하지 않게

### 좌: 도넛 게이지 (DaisyDisk 영감, 단순화)
- 전체 디스크 사용률. 색은 임계치 기반 (70% 노랑, 85% 주황, 95% 빨강).
- 호버하면 카테고리별 비율이 도넛 링으로 분해됨 (선택적 상호작용).

### 우상: 핵심 지표 + CTA
- "267 GB / 492 GB" 큰 숫자. 장식 없음.
- **"AI 분석 시작하기"** 버튼이 가장 눈에 띄는 CTA. 우리 차별점.

### 중앙: 카테고리 카드 (CleanMyMac 영감, 승인 기반)
- 5-6개 카테고리 나열. 클릭하면 해당 카테고리만 상세 뷰.
- 각 카드에 막대 (전체 대비 비율) + 아이콘 + 용량.
- ❌ "원클릭 정리" 버튼 안 붙임 (블랙박스 방지).

### 하단: Top 항목 (OmniDiskSweeper 영감, 설명 추가)
- 현재 구현된 Top 30. 단, 각 항목에 **컨텍스트 한 줄** 추가:
  - "마지막 수정 3개월 전"
  - "개발 프로젝트 — 빌드 아티팩트 8GB 포함"
  - "중복 파일 42개"
- 클릭하면 드릴다운 시트 (OmniDiskSweeper 컬럼 뷰).
- 우클릭 → Finder에서 표시 / 휴지통 이동 / AI 분석에 포함.

## 차후 덧댈 것 (Chunk 7+)

- **트리맵 뷰 토글** — GrandPerspective 영감, "개발환경" 탭에서 시각적 드릴다운
- **메뉴바 미니 게이지** — 항상 사용률 표시
- **비교 뷰** — "지난 달 대비 +12 GB" (히스토리 기반)

## Visual Language

- **Typography**: 시스템 폰트, 숫자는 `monospacedDigit()`
- **Color**: 카테고리별 색상은 이미 `StorageCategory.color`에 정의됨. 배경은 `.regularMaterial` 혹은 `GroupBox` 표준.
- **Spacing**: 24px outer padding, 12-16px inner, 카드 간 16px
- **Depth**: 카드에 subtle shadow 대신 `.background(.regularMaterial)` + `.clipShape(RoundedRectangle(cornerRadius: 12))`
- **Icons**: SF Symbols only (네이티브 일관성)
- **Motion**: 카테고리 카드 hover → scale 1.02, 숫자 변경 → `.contentTransition(.numericText())`
